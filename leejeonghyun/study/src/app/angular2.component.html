Generate Component

jhLee$ ng g c testComponent --flat --is --it --spec false

ng new [projectName]명령어로 생성된 directory 내에서 타이핑 해야 한다.

우선 --로 시작하는 부분은 전부 flag이며 component를 생성하는데 옵션을 부가한다.

기본적으로 아무 옵션도 주지 않았을때 projectName.component.css, projectName.component.html, projectName.component.ts projectName.component.spec.ts 4개의 파일이 projectName directory와 함께 생성된다.

flag option
--flat directory 생성 안함
--is css파일 생성 안함
--it HTML파일 생성 안함
--spec false spec.ts파일 생성 안함
